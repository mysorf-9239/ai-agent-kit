# Skills

This directory stores executable skills.

Current baseline skills:
- `validate-dti-dataset`
- `train-dti-model`
- `evaluate-dti-model`

Rules:
- every skill directory MUST contain `SKILL.md`
- long technical detail MUST move to `references/`
- reusable snippets and manifests MUST move to `assets/`
- skill outputs MUST align with `templates/` and `state/`
