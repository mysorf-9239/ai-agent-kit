# AI Agent Kit

Production-oriented Agent Skill Kit for machine learning, bioinformatics, and drug-target interaction workflows.

Primary directories:
- `guidelines/`
- `orchestrator/`
- `skills/`
- `workflows/`
- `templates/`
- `state/`
- `manifests/`
- `artifacts/`
- `reports/`
- `scripts/`

Suggested starting order:
1. read `guidelines/README.md`
2. inspect `orchestrator/dti-orchestrator.md`
3. inspect `workflows/dti-benchmark-training/WORKFLOW.md`
4. inspect `skills/`
5. run `python scripts/run_dti_demo.py`
