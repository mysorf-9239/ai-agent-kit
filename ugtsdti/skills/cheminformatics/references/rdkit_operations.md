# Cheminformatics & RDKit Operations

## 1. SMILES (Simplified Molecular Input Line Entry System)

**Format rules:**
- Atoms: element symbols (C, N, O, S, F, Cl, Br, I)
- Aromatic atoms: lowercase (c, n, o)
- Branches: parentheses `()`
- Rings: numbers `C1CCCCC1` (cyclohexane)
- Double bond: `=`, Triple: `#`

**Canonical vs non-canonical SMILES:**
- Same molecule can have many valid SMILES strings
- Canonical SMILES: unique representation generated by RDKit
- Always use `Chem.MolToSmiles(mol)` for canonical form

## 2. Molecular Fingerprints & RDKit Operations

### Fingerprint Types

| Fingerprint | Dimensions | Description | Best for |
|---|---|---|---|
| **Morgan (ECFP)** | 2048 | Circular substructures, radius r | Drug similarity, DTI |
| **MACCS Keys** | 166 | SMARTS-defined structural keys | Scaffold analysis |
| **RDKit FP** | 2048 | Path-based substructures | General purpose |

*Note: ECFP4 is roughly equivalent to Morgan fingerprint with radius=2.*

### Common RDKit Snippets

```python
from rdkit import Chem
from rdkit.Chem import AllChem, Descriptors

# 1. Parse and Canonicalize SMILES
smiles = "CC(=O)Oc1ccccc1C(=O)O"  # Aspirin
mol = Chem.MolFromSmiles(smiles)
if mol is None:
    print("Invalid SMILES!")
canonical_smiles = Chem.MolToSmiles(mol)

# 2. Compute Morgan Fingerprint (Bit Vector)
# Use GetMorganFingerprintAsBitVect (0/1 values, used for Tanimoto)
fp_bit = AllChem.GetMorganFingerprintAsBitVect(mol, radius=2, nBits=2048)

# 3. Calculate basic properties
mw = Descriptors.MolWt(mol)
logp = Descriptors.MolLogP(mol)
n_heavy_atoms = mol.GetNumHeavyAtoms()
```

## 3. Molecular Graph Representation (For GNNs)

Used in GNN-based models (Teacher branch of UGTS-DTI):
- **Nodes**: atoms (with atom features)
- **Edges**: bonds (with bond type features)

### PyTorch Geometric Graph Conversion
```python
from torch_geometric.data import Data
import torch

def smiles_to_graph(smiles: str) -> Data:
    mol = Chem.MolFromSmiles(smiles)
    
    # 1. Node features (AtomicNum, Degree, etc.)
    atom_features = []
    for atom in mol.GetAtoms():
        feats = [
            atom.GetAtomicNum(),
            atom.GetDegree(),
            atom.GetFormalCharge(),
            int(atom.GetIsAromatic())
        ]
        atom_features.append(feats)
    x = torch.tensor(atom_features, dtype=torch.float)
    
    # 2. Edge index
    edge_index = []
    for bond in mol.GetBonds():
        i, j = bond.GetBeginAtomIdx(), bond.GetEndAtomIdx()
        edge_index += [[i, j], [j, i]]  # undirected
        
    edge_index = torch.tensor(edge_index, dtype=torch.long).t().contiguous()
    return Data(x=x, edge_index=edge_index)
```
